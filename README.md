# Frask Estruturas - Landing Page

Landing page desenvolvida para uma empresa de estruturas para eventos corporativos, sociais e promocionais.

O objetivo do projeto é apresentar serviços, portfólio e facilitar o contato direto via WhatsApp.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (básico)
- SVG Icons
- Google Fonts

## Estrutura do projeto

frask-site

├── index.html

│ └── responsive.css
│
├── js
│ └── script.js
│

│
└── README.md

## Funcionalidades

- Layout responsivo
- Navegação entre seções
- Botão de contato direto via WhatsApp
- Estrutura organizada de CSS
- Ícones SVG otimizados

## Objetivo do projeto

Este projeto foi desenvolvido com foco em:

- boas práticas de HTML semântico
- organização de CSS em múltiplos arquivos
- estrutura de projeto front-end profissional
- uso em portfólio de desenvolvedor
